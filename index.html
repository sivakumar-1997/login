Scenarios involving links with no Title Attributes

Pickup Scenario: no href , no linktext, no title (link will not be visible but might show up on screenreader)


Autofix: should be assigned aria-hidden true (link will be hidden from screenreader)

Autofix: in-page href should be provided (link will look like & behave like a link)

Autofix: extract title from href (the link provides more context for the link destination)
